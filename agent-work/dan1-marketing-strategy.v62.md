## 2. The 3-tier moat ladder (unchanged from v57)

| Tier | Asset | Compounding | Time to build |
|------|-------|-------------|---------------|
| **Tier 1: Code & trust** | The 7 daemons, MIT-licensed, 5 live today | Stars, forks, citations | 0-90 days |
| **Tier 2: Hardware reference design** | BOM target, sub-50g, MIT schematic | Manufacturing partners, OEM contracts | 90-365 days |
| **Tier 3: Distribution** | First 1,000 daily-active users, first 50 dev-kit alpha units | Press, word-of-mouth, platform integration | 365-730 days |

**The compounding bet:** Tier 1 is a bet that GitHub stars + forks + a 5/7 daemons-live "stack" page converts to Tier 2. Tier 2 is a bet that the BOM target holds and a wearable OEM signs an LOI. Tier 3 is a bet that the first 1,000 users are loud enough to ride a press cycle.

**The risk:** Tier 1 is shipping now. Tier 2 is a Q3-Q4 bet. Tier 3 is a Q1-Q2 2027 bet. The marketing strategy has to land all three before the Apple-window closes in ~14 months.
