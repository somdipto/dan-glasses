## WHAT IT DOES (Section 4)

### Section heading (H2)

```
What Dan Glasses actually does.
```

### Three-column feature grid

#### Column 1: 👁️ **Sees**

```
Salience-gated vision. The camera only runs inference when motion or a face is detected.

Default state: silent.
Active state: sub-250ms caption.

Powered by LFM2.5-VL-450M (Liquid AI). 209MB. MIT weights.
```

#### Column 2: 👂 **Listens**

```
Push-to-talk audio. The microphone only listens when you ask.

Default state: muted.
Active state: whisper.cpp transcription, sub-300ms.

Runs offline. No data leaves your device.
```

#### Column 3: 🧠 **Remembers**

```
Persistent semantic memory. Every conversation, every observation, stored locally.

Episodic (what happened when).
Semantic (what it means).
Procedural (how to do things).

SQLite + all-MiniLM-L6-v2 vectors. 384-dim. Stays on disk.
```

#### Column 4: 🗣️ **Speaks**

```
On-device TTS, not cloud. KittenTTS, <25MB, CPU-friendly.

Default state: silent.
Active state: <500ms short responses.

No API call. No "this message was generated by AI" voice.
```

#### Column 5: 🔧 **Acts**

```
8 core Paperclip agent workflows. Expense reports, email drafting, Notion updates, PDF generation, Slack posting, calendar booking, meeting summaries, travel planning.

All in your own Docker container. Never on our servers.
```

#### Column 6: 🔒 **Stays private**

```
0 cloud calls. 0 data uploads. 0 telemetry. 0 lock-in.

Your AI, in your container, on your face. Not Meta's. Not Alibaba's. Yours.
```

Pillar count: 5 (Proactive + Local + OSS + India + AGI) ✅
